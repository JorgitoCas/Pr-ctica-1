Página de exposicion de la Web Storage API




Primero se tiene que saber que esta api funciona en los principales buscadores web disponibles en el mercado

método Set Item

se usa la estructura localStorage.setItem("el tipo que guardas, es como una id", "lo que se guarda");

se veria algo así localStorage.setItem("name", "JorgitoCas");



Y después está el

getItem

que es para recuperar el valor almacenado, esto se puede usar para guardar nombres y buscarlos despues y es algo asi: localStorage.getItem("name");



sessionStorage , este solo guarda el dato por esa seción, si se cierra la pestaña, se pierde

por ejemplo

en una etiqueta p usaré un id para imprimir el dato guardado

se veria algo así: sessionStorage.setItem("nombre","Jorgito Cas"); document.getElementById("demostracion").innerHTML = sessionStorage.getItem("nombre");



setItem() Method

Este guarda en el almacen un dato, se ve así: sessionStorage.setItem("nombre", "Jorgito Doe");


y así mismo getItem() busca el dato en el almacen, se ve así: sessionStorage.getItem("nombre");
